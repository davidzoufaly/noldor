description: Noldor milestone — draft/activate/edit/list milestones
---

Run the Noldor milestone flow. Read `docs/noldor/milestones.md`, then use
`pnpm noldor milestone` to draft/activate/edit/list a milestone at
`docs/milestones/<slug>.md`. Vision's `current-milestone:` frontmatter points at the
active one. Milestones are optional — the framework works without one.
