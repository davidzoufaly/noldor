description: Noldor new-feature — scaffold a blank feature MD
---

Run the Noldor new-feature flow for work not in the roadmap/backlog. Read
`docs/noldor/feature-md-schema.md`, then scaffold `docs/features/<slug>.md` with the
required frontmatter + body sections via the `pnpm noldor` features verb. For a
roadmap/backlog entry, use promote instead.

Commit messages need a `Noldor-FD: <slug>` trailer.
